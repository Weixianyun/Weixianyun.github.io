在搭建好 Hexo 博客的基础框架后，为了追求极致干净、类似 Chrome 浏览器的现代极简质感，我将网站主题切换为了 Keep。

这篇文章记录了如何提取 Keep 主题的底层配置，并将其替换为我专属的全局标题与 JPG 格式的个人 Logo。

1. 提取主题的“原厂说明书”

使用 npm 安装 Keep 主题后，配置文件默认深藏在系统目录中。为了安全修改且防止日后升级被覆盖，我们需要将其提取到根目录：

  1. 进入路径：node_modules/hexo-theme-keep/
  2. 找到 _config.yml 文件并复制。
  3. 将其粘贴到博客的根目录（与 source 同级），并重命名为 _config.keep.yml。

2. 修改全局信息与自定义 Logo

准备好专属的头像和标志（例如 logo.jpg），将其放置在 source/images/ 目录下。

打开刚才提取的 _config.keep.yml，定位到 base_info 区域进行如下修改：

base info:
  # Theme primary color
  primary color: "#0066cc"

  # Blog website title
  title: Weixianyun

  # Blog website author name
  author: Weixianyun

  # Author avatar
  avatar: /images/logo.jpg

  # Website LOGO
  logo: /images/logo.jpg

  # Website favicon
  favicon: /images/logo.jpg
